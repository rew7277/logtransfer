# ObserveX V3 SaaS Edition

A deployable Flask starter for an AI-native observability and secure log intelligence platform.

## Features
- Landing page
- Sign in / Create workspace
- SQLite-backed auth and workspace model
- Demo SaaS dashboard
- Log Explorer
- Trace Explorer
- Alerts Center
- Security / Masking
- Settings
- Premium dark UI

## Local run
```bash
pip install -r requirements.txt
python app.py
```

## Railway
- Deploy as a Python service
- Start command: `python app.py`
- Port is read from `PORT`

## Demo login
Create a workspace from `/register`, then sign in.
