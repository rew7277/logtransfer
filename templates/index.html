POST /ingest/json

Send JSON logs over HTTP — single, array, or NDJSON.

Single event

curl -X POST https://your-domain/ingest/json \
  -H "Authorization: Bearer oxk_..." \
  -H "Content-Type: application/json" \
  -d '{"timestamp":"2024-01-15T10:30:00Z","level":"error","source":"payment-service","message":"Timeout"}'

Array (batch)

curl -X POST https://your-domain/ingest/json \
  -H "Authorization: Bearer oxk_..." \
  -H "Content-Type: application/json" \
  -d '[{"message":"Started"},{"message":"Done","level":"success"}]'

NDJSON

{"message":"line 1","level":"info"}
{"message":"line 2","level":"warn"}
{"message":"line 3","level":"error"}

POST /ingest/file

Upload .log .txt .json .ndjson .csv files.

Single file

curl -X POST https://your-domain/ingest/file \
  -H "Authorization: Bearer oxk_..." \
  -F "file=@/path/to/app.log"

Multiple files

curl -X POST https://your-domain/ingest/file \
  -H "Authorization: Bearer oxk_..." \
  -F "files=@app.log" -F "files=@error.log"

Python

import requests
resp = requests.post(
    "https://your-domain/ingest/file",
    headers={"Authorization": "Bearer oxk_..."},
    files={"file": ("app.log", open("app.log","rb"), "text/plain")},
)
print(resp.json())
