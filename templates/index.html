POST /ingest/json

Send JSON logs over HTTP — no file needed.

Single event:

curl -X POST https://your-domain/ingest/json \
  -H "Authorization: Bearer oxk_..." \
  -H "Content-Type: application/json" \
  -d '{"timestamp":"2024-01-15T10:30:00Z","level":"error","source":"payment-service","message":"Timeout connecting to gateway","requestId":"req-abc123"}'

Batch (array):

curl -X POST https://your-domain/ingest/json \
  -H "Authorization: Bearer oxk_..." \
  -H "Content-Type: application/json" \
  -d '[{"message":"Started"},{"message":"Completed","level":"success"}]'

NDJSON (newline-delimited):

{"message":"line 1","level":"info"}
{"message":"line 2","level":"warn"}
{"message":"line 3","level":"error"}

POST /ingest/file

Upload log files — .log, .txt, .json, .ndjson, .csv

Single file:

curl -X POST https://your-domain/ingest/file \
  -H "Authorization: Bearer oxk_..." \
  -F "file=@/path/to/app.log"

Multiple files:

curl -X POST https://your-domain/ingest/file \
  -H "Authorization: Bearer oxk_..." \
  -F "files=@app.log" \
  -F "files=@error.log" \
  -F "files=@access.json"

Python SDK example:

import requests

API_KEY = "oxk_..."
with open("app.log", "rb") as f:
    resp = requests.post(
        "https://your-domain/ingest/file",
        headers={"Authorization": f"Bearer {API_KEY}"},
        files={"file": ("app.log", f, "text/plain")},
    )
print(resp.json())
