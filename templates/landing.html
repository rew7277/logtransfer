FAQ

Common implementation questions

Why separate sign in and create account?

Because that is the standard SaaS flow. It improves clarity and feels more trustworthy for enterprise users.

Why don’t subdomains show automatically?

App code stores the organization slug, but real tenant URLs also need wildcard DNS and Railway custom domain routing.

Can admins create users inside the organization?

Yes. The first user is the admin, and admins can invite or create more users later.
