Features

Everything you need to observe your systems

A complete platform — not just a log viewer. Search, alert, schedule, and team up.

📥
Multi-format Ingestion
Push logs as JSON, CSV, plain text, MuleSoft block format, or upload files directly. Auto-detection picks the right parser every time.
☁️
S3 Integration
Connect any S3-compatible bucket. ObserveX polls your bucket on a schedule, downloading and parsing new log files automatically.
🔑
API Key Management
Create scoped API keys with prefix-based lookup and LRU caching. Revoke instantly. Every key access is tracked with last-used timestamps.
🔔
Smart Alerts
Define threshold-based alert rules on any field. Get notified via email, Slack webhook, or Teams webhook the moment a rule fires.
⏱
Scheduled Jobs
Set up cron-based ingestion jobs that pull from S3 or API sources automatically. View next-run estimates and trigger manually anytime.
📊
Dashboard & Timeseries
Real-time summary cards showing error rates, ingestion volume, top sources, and a per-minute timeseries chart of log activity.
🔍
Log Search & Filter
Filter by level (ERROR, WARN, INFO, DEBUG), source, date range, and free-text search. Click any log to expand full metadata.
👥
Team & Invitations
Invite team members by email with role-based access (admin / viewer). Accept via invite link. Admins can manage users, disable admin-only mode.
🏢
Multi-workspace
Every organization gets its own isolated workspace with a unique slug URL. Branding, theme color, and logo text are fully customizable.
📋
Audit Log
Every action — logins, ingestion, user changes, key creation — is recorded in an immutable audit trail with timestamps and actor info.
📈
Observability Metrics
Track ingestion rates, active integrations, error trends, and alert history across all sources in your workspace observability view.
🔐
Enterprise Security
Rate limiting on all auth endpoints, SSRF-protected URL validation, security headers (CSP, HSTS), bcrypt passwords, and CSRF tokens built in.
How it works

From zero to observability in minutes

01

Create your workspace

Register with your organization name and email. Your workspace gets a unique URL — yourco.observex.in — ready instantly, no email verification needed.

02

Connect a log source

Generate an API key and POST logs directly, connect an S3 bucket, or upload a log file. ObserveX handles JSON, CSV, plain text, and MuleSoft formats automatically.

03

Search & set alerts

Use the live log viewer to filter by level, source, and time. Define alert rules on any field threshold — get notified via Slack, Teams, or email.

04

Automate with scheduled jobs

Set up cron jobs to pull from S3 or APIs on a schedule. View ingestion history, trigger manually, and monitor all jobs from the dashboard.
